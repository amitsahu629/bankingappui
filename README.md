# 💰 BankingApp UI

A responsive and user-friendly frontend for a modern **Online Banking Application**, developed using **Angular**. This project is built to provide seamless interaction for retail users to manage their banking needs — including account management, transactions, history, and authentication.

🚀 **Live Demo**: [View App](https://amitsahu629.github.io/bankingappui)

---

## 📌 Features

- ✅ Clean and responsive UI with Angular
- 🔐 User authentication (Login/Signup)
- 🧾 View account details and transaction history
- 💸 Initiate and track fund transfers
- 📊 Dashboard for personal finance overview
- 🕵️ Role-based navigation (Customer/Admin)
- 🎨 Smooth animations and modern design using Angular Material & Bootstrap

---

## 🛠️ Tech Stack

- **Frontend:**  HTML5, CSS3
- **Styling:** Bootstrap 5
- **State Management:** LocalStorage / SessionStorage

## 📦 Installation

1. **Clone the repo**
   ```bash
   git clone https://github.com/amitsahu629/bankingappui.git
   cd bankingappui
````

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Run the Angular app**

   ```bash
   ng serve
   ```

4. Visit: `http://localhost:4200/`
